# Excel Sales Analysis Project

## Project Overview
This project analyzes sales data using Microsoft Excel.

## Analysis Performed
- Total Sales Calculation
- Region-wise Sales Analysis (Pivot Tables)
- Product-wise Sales Analysis
- Identification of Highest and Lowest Performing Regions
- Business Insight Extraction

## Tools Used
- Microsoft Excel
- Pivot Tables
- XLOOKUP / VLOOKUP
- Conditional Formatting

## Key Insights
- Identified revenue concentration patterns
- Compared regional performance
- Evaluated product-level sales contribution

This project demonstrates Excel-based data analysis skills suitable for entry-level analyst roles.
